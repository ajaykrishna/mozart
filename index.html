Mozart

Mozart provides an intuitive way to compose and deploy IoT applications. It features a web interface to define a composite IoT service, verify its compatibility and deploy the service

Click-and-Play compatibility Checks and Deployment

Build and verify of your IoT composition in a few clicks! Simply select the objects and define your bindings to check for compatibility. Mozart does the job of automated analysis and formal verification behind the scenes and displays the results.

Design time Checks

Compatibility check helps in early detection of errors while designing an IoT service

Reliable Deployment

Majord'Home a proprietary platform for SDN-based deployments takes care of network deployments. A deployment plan is computed for each composition to ensure reliable deployment.

Interface based modelling

Mozart is based on abstraction of object behaviour into an interface based model. Easy to understand yet, encapsulates the complete behaviour of objects

UI based composition

Intuitive web-interface guides the user to build composite services for smart home

Extend

In addition to the web interface, Mozart provides REST API for clients to build custom interfaces suited to their needs.
